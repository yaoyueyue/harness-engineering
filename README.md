# Harness Engineering

A reusable scaffold for **autonomous engineering work** — specs, templates, scripts, state files, and human checkpoints — built so the *system* (not the prompt) owns the agent loop.

🌐 **Live demo:** https://yaoyueyue.github.io/harness-engineering/

## What this is

The shift is from **prompt engineering** to **agentic engineering** — building the scaffold the agent runs *inside*. With the scaffold, automation becomes repeatable, parallel, and auditable.

See the [live demo](https://yaoyueyue.github.io/harness-engineering/) for the full architecture: lifecycle, components, human checkpoints, and design principles.

## Status

This repo currently hosts the public demo page. The full harness implementation (specs, templates, validator/orchestrator/porter agents, scripts) will be open-sourced here in a future commit.
